PNopt is a software package that uses proximal Newton-type methods to solve convex optimization problems in composite form. For details about these methods, please refer to our paper.

Usage

PNopt shares a similar interface with TFOCS, a popular software package for solving convex optimization problems in composite form. PNopt is also compatible with the suite of function generators included with TFOCS. To call PNopt, use:

[x, f, output] = PNopt(smoothF, nonsmoothF, x0, options);

The input arguments are

  • smoothF: a smooth function
  • nonsmoothF: a nonsmooth function
  • x0: a starting point for the solver
  • options: an options structure created using the PNoptimset function
